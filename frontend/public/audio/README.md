Optional lottery sound assets go here.

Supported path used by the prize wheel:
- `lottery-tick.mp3`
- `lottery-win.mp3`

When `frontend/public/audio/lottery-tick.mp3` exists, the survey lottery page will try to loop it during wheel spins.
When `frontend/public/audio/lottery-win.mp3` exists, the survey lottery page will try to play it after the winner is revealed.
